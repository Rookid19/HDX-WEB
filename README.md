## Getting Started

>> clone the repository to get started , you must also have expo cli already installed

### Installation

installation for both npm and yarn
> after clonning repository
> cd into the main directory then 
* npm
  ```sh
  npm install
  ```
  or 
  
* yarn
  ```sh
  yarn
  ```
